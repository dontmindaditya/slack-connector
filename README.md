# 🚀 Slack Connector (Collectium Integration)

A modular **Slack MCP Connector** built using **FastAPI + Supabase**, designed to integrate Slack communication into the Collectium platform.

This connector enables seamless interaction with Slack including message reading, writing, and real-time event syncing.

---

## ✨ Features

- 🔐 Slack OAuth 2.0 Authentication  
- 💬 Send messages to Slack channels  
- 📥 Fetch messages from Slack channels  
- 🔄 Sync Slack messages to Supabase  
- 📡 Slack Events Webhook (real-time updates)  
- ⚡ Async & scalable backend (FastAPI)  
- 🧩 Clean and modular architecture  

---

## 🛠️ Tech Stack

- **Backend:** FastAPI (Python)  
- **Database:** Supabase  
- **Slack Integration:** slack_sdk  
- **HTTP Client:** httpx
